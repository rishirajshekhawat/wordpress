== Description ==

EasyPost Shipping Solution is a simple shipping API for your app or site. We believe in making purchasing labels, tracking packages, checking rates, verifying addresses, and all other shipping related functions as easy as possible.



== Installation ==

1. Install the Woocommerce EasyPost Shipping Tool Plugin 
2. Activate the plugin
3. Go to the WooCommerce Easypost Config 
4. Check the Enable/Disable Checkbox.
5. Enter the settings that you would like you to use
6. Enter the dimensions of the products, (Easypost uses this to determine suitability of products)
